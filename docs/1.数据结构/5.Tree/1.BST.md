# Binary Search Tree

## 简介

### 概念

- 二分搜索树的每个节点的值:
  - 大于其左子树的所有节点的值
  - 小于其右子树的所有节点的值
- 二叉树不一定是“满”的



### 复杂度

| 方法        | 复杂度 |
| ----------- | ------ |
| Add         | O(h)   |
| Delete      | O(h)   |
| Max         | O(h)   |
| Min         | O(h)   |
| Predecessor | O(h)   |
| Succesor    | O(h)   |
| Seach       | O(h)   |
